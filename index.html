1. Squeenote

    A way for nerds to present to other nerds

    Press ← or → on the keyboard to move between slides, or use the buttons at the top right.

    http://github.com/danski/squeenote

  3. Write presentations with simple HTML5

    <!DOCTYPE html>
<html lang="en">
<head>
  <title>My presentation</title>
  ...
</head>
<body>
  <ol>
    <li>
      <h2>Write presentations with simple HTML5</h2>
      <code><pre>
        &lt;!DOCTYPE html&gt;
        &lt;html lang=&quot;en&quot;&gt;
        ...
        &lt;/html&gt;
      </pre></code>
    </li>
  </ol>
</body>
</html>

  4. Keep everyone in sync with Presenter Follow

    Audience members can navigate slides at will and rejoin you as they please

  5. Theme it however you like

    • Squeenote's opinionated (but simple) markup lets you apply your own styles and transitions.
    • You can even override the Squeenote Bar's styles if you so choose
    • Squeenote's default theme uses CSS3 transitions for animation
    • Squeenote publishes events that your theme can use for transitions and styling.

  6. Deploy a presentation for remote viewing, or run it locally for a small crowd

    node squeenote.node.js -f mypresentation.html -p mypresenterpassword -P 8080

  7. Upload them as static files for later viewing

    All static file URLs are still valid

  8. Open-source soup

    • Squeenote Server uses Node.js and Socket.io
    • Squeenote UI uses jQuery
    • Squeenote Client uses Socket.io for server connectivity
